# 卡密数据自动备份系统

## 📋 功能

- ✅ 导出所有卡密到JSON文件
- ✅ 保留最近10次备份
- ✅ 可选提交到GitHub
- ✅ 支持一键恢复

## 🚀 使用方法

### 手动备份

双击运行: `手动备份.bat`

### 命令行备份

```bash
# 普通备份
node backup-cardkeys.js

# 备份并提交到GitHub
node backup-cardkeys.js --git
```

### 在服务器中自动备份

修改 `server.js`,在添加卡密后自动备份:

```javascript
// 在添加卡密成功后
const { exportToJSON } = require('./backup-cardkeys');
exportToJSON();
```

## 📁 备份文件位置

```
验证服务器升级版/
└── backups/
    ├── cardkeys-backup-2026-07-20T10-30-00.json
    ├── cardkeys-backup-2026-07-20T11-00-00.json
    └── ...
```

## 🔄 恢复备份

```javascript
const { restoreFromBackup } = require('./backup-cardkeys');
restoreFromBackup('./backups/cardkeys-backup-2026-07-20T10-30-00.json');
```

## 🎯 建议

1. **每次生成新卡密后立即备份**
2. **定期备份到GitHub**
3. **保存备份文件到多个位置**
4. **考虑升级到付费版或使用PostgreSQL**
